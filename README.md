# timer
A no-nonsense nodejs timer

[![npm version](https://badge.fury.io/js/%40iwharris%2Ftimer.svg)](https://badge.fury.io/js/%40iwharris%2Ftimer)

## Usage

```javascript
const Timer = require('@iwharris/timer');

const t = new Timer();

t.elapsedMilliseconds();

// 3421
```
